USAGE:

molality  =  gsw_molality(SA)

DESCRIPTION:

Calculates the molality of seawater.

INPUT:

SA  =  Absolute Salinity                                        [ g/kg ]

OUTPUT:

molality  =  molality of seawater                             [ mol/kg ]

EXAMPLE:

SA = [34.7118; 34.8915; 35.0256; 34.8472; 34.7366; 34.7324;]
molality  = gsw_molality(SA)
molality =
 1.145084756351415
 1.151227076302664
 1.155812234675510
 1.149712646992389
 1.145932308325875
 1.145788768234685
