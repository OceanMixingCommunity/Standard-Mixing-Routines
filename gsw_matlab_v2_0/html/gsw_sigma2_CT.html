USAGE:

sigma2_CT = gsw_sigma2_CT(SA,CT)

DESCRIPTION:

Calculates potential density anomaly with reference sea pressure of
2000 dbar.
Click for a more detailed description of potential
density anomaly with pr of 2000 dbar (sigma_2).

INPUT:

SA  =  Absolute Salinity                                        [ g/kg ]
CT  =  Conservative Temperature                                [ deg C ]
SA & CT need to have the same dimensions.

OUTPUT:

sigma2_CT  =  potential density anomaly with                  [ kg/m^3 ]
              respect to a reference pressure of 2000 dbar,
              that is, this potential density - 1000 kg/m^3.

EXAMPLE:

SA = [34.7118; 34.8915; 35.0256; 34.8472; 34.7366; 34.7324;]
CT = [28.8099; 28.4392; 22.7862; 10.2262;  6.8272;  4.3236;]
sigma2_CT = gsw_sigma2_CT(SA,CT)
sigma2_CT =
 30.023190232854176
 30.283832597402807
 32.265717400800895
 35.474625433103029
 36.067249427311708
 36.492547357788226
